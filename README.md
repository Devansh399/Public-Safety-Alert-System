
# Public Safety Alert System

The Public Safety Alert System is a full-stack web application designed to improve emergency response and public awareness. It enables users to report incidents, receive real-time safety alerts, view nearby emergencies on an interactive map, and access AI-powered risk analysis based on reported incidents. The system is built with scalability and real-world deployment in mind, making it suitable for government agencies, emergency services, and smart city initiatives.

## Features
- 🚨 Real-time emergency alerts
- 📍 Incident reporting with location
- 🗺️ Interactive map for nearby incidents
- 🤖 AI-powered incident severity prediction
- 📊 Safety analytics dashboard
- 🔔 Notification system
- 👤 Secure user authentication
- 🛠️ Admin panel for incident management
